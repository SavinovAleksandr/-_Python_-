# Начало работы

## Установка Python

### На Mac:
1. Проверьте, установлен ли Python:
   ```bash
   python3 --version
   ```

2. Если не установлен, установите через Homebrew:
   ```bash
   brew install python3
   ```

### На Windows:
1. Скачайте Python с [python.org](https://www.python.org/downloads/)
2. При установке отметьте "Add Python to PATH"
3. Проверьте установку:
   ```bash
   python --version
   ```

## Установка зависимостей

### На Mac (для работы с Excel):
```bash
cd "/Users/asavinov/МАКРОС_ДУ/Обучение Python"
pip3 install -r requirements.txt
```

### На Windows (дополнительно для COM):
```bash
pip install pywin32
```

## Структура работы

### На Mac:
1. Изучайте материалы в папке `Уроки/`
2. Выполняйте задания в папке `Задания/`
3. Запускайте примеры:
   ```bash
   python3 "Уроки/Урок_01_Основы_Python/пример_01_переменные.py"
   ```

### На Windows:
1. Синхронизируйте код через GitHub
2. Установите все зависимости (включая pywin32)
3. Тестируйте код с RasterWin

## Запуск Python скриптов

### На Mac/Linux:
```bash
python3 имя_файла.py
```

### На Windows:
```bash
python имя_файла.py
```

## Работа с заданиями

1. Откройте файл задания (например, `Задание_1.1_Переменные.py`)
2. Найдите комментарий `# ===== ВАШ КОД НИЖЕ =====`
3. Напишите свой код вместо комментариев
4. Запустите скрипт для проверки
5. Исправьте ошибки при необходимости

## Получение помощи

Если что-то не работает:
1. Проверьте синтаксис (отступы, скобки, кавычки)
2. Прочитайте сообщение об ошибке внимательно
3. Посмотрите примеры в папке `Уроки/`
4. Используйте `print()` для отладки
