# Урок 5: Расчеты и циклы

## Цель урока
Реализовать логику определения последнего устойчивого режима через бинарный поиск.

## Содержание урока

### 1. Бинарный поиск (дихотомия)

См. `Примеры/пример_цикл_устойчивости.py` - там есть подробные примеры.

### 2. Цикл определения устойчивости

Основная логика:
1. Устанавливаем границы поиска (минимум и максимум)
2. Вычисляем среднюю точку
3. Проверяем устойчивость в средней точке
4. Сдвигаем границы в зависимости от результата
5. Повторяем до достижения нужной точности

## Примеры

Смотрите `Примеры/пример_цикл_устойчивости.py` - там полные примеры бинарного поиска.

- `Задание_5.2.py` - Реализация бинарного поиска
