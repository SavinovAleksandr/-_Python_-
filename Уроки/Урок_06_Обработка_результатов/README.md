# Урок 6: Обработка результатов

## Цель урока
Научиться извлекать данные из протоколов RasterWin и записывать результаты в Excel.

## Содержание урока

### 1. Извлечение данных из протокола

См. `Примеры/пример_чтение_протокола.py`

### 2. Работа со словарями для структурирования данных

```python
результаты = {
    "режим": "установившийся",
    "напряжение": 220.5,
    "мощность": 1000.0,
    "устойчивость": True
}
```

### 3. Запись результатов в Excel

См. `Примеры/пример_запись_результатов.py`

- `Примеры/пример_чтение_протокола.py` - извлечение данных
