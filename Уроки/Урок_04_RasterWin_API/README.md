# Урок 4: RasterWin API

## Цель урока
Научиться работать с методами RasterWin API для запуска расчетов установившегося и переходного режимов.

## Содержание урока

### 1. Запуск расчета установившегося режима (RGM)

```python
# Базовый вызов
код_результата = rastr.rgm()

# С параметрами
код_результата = rastr.rgm("p")  # "p" - плоский старт

# Проверка результата (0 = успех)
if код_результата == 0:
    print("✅ Режим сбалансирован")
else:
    print("❌ Режим не сбалансирован")
```

### 2. Запуск динамического расчета

```python
# Получение объекта динамики
fw_dynamic = rastr.FWDynamic()

# Запуск расчета (Run-режим)
результат = fw_dynamic.Run()

# Проверка устойчивости
if fw_dynamic.SyncLossCause == 0:
    print("✅ Система устойчива")
else:
    print("❌ Система неустойчива")
```

### 3. Чтение результатов из протокола

```python
# Результаты динамического расчета
время = fw_dynamic.TimeReached
сообщение = fw_dynamic.ResultMessage
причина_потери = fw_dynamic.SyncLossCause
```
