# Решения заданий

**ВНИМАНИЕ**: Эта папка содержит решения заданий.

**Рекомендация**: 
- Сначала попробуйте решить задания самостоятельно
- Используйте решения только для проверки или если совсем не получается
- Изучите решения, чтобы понять правильный подход

## Структура решений

- `Решение_1.1_Переменные.py` - решение задания 1.1
- `Решение_1.2_Списки.py` - решение задания 1.2
- `Решение_1.3_Словари.py` - решение задания 1.3
- `Решение_1.4_Функции.py` - решение задания 1.4

## Как использовать

1. Попробуйте решить задание самостоятельно
2. Запустите свое решение и проверьте результат
3. Если не получается, посмотрите решение
4. Разберитесь, почему решение работает именно так
5. Попробуйте решить снова, не глядя в решение
